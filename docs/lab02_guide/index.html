สถานที่แนะนำ (Highlights)

ปัดรูปภาพเพื่อชมบรรยากาศเพิ่มเติม

1. บึงสีฐาน (Sithan Lake)

จุดเช็คอินยอดนิยมสำหรับชมพระอาทิตย์ตกและออกกำลังกาย ยามเย็นแสงสวยมาก เหมาะแก่การถ่ายรูปพอร์ตเทรต

2. หอศิลปวัฒนธรรม

แหล่งเสพงานศิลป์ริมบึง สถาปัตยกรรมสวยงาม มีนิทรรศการหมุนเวียนให้เข้าชมฟรีตลอดทั้งปี

3. พิพิธภัณฑ์ธรรมชาติวิทยา

สนุกกับโลกดึกดำบรรพ์และไดโนเสาร์ภูเวียง ภายในจัดแสดงแสงสีเสียงน่าตื่นตาตื่นใจ

4. อุทยานวิทยาศาสตร์

ตึกที่ทันสมัยที่สุดใน มข. มีร้านกาแฟ Amazon วิวสวย และ Co-working space สำหรับนั่งทำงานชิลๆ
